# fc3d-public-draw-feed

福彩3D 客户版**签名开奖更新**的公开只读分发仓库。

- 本仓库仅用于**公开分发**经过签名的福彩3D开奖更新数据（`latest.json` 及 `issues/<期号>.json`）。
- 数据由**自动化流程**生成并发布，**不接受人工修改开奖数据**。
- 每个开奖更新包都带有 **Ed25519 数字签名**；客户端在导入前会**校验签名**，签名不通过的数据将被拒绝，不会写入本地。
- 本仓库**不包含**任何密钥、凭证、授权信息、客户资料或业务数据；仅包含公开开奖字段与其签名。

> 号码仅用于结构化研究与记录，不预测、不保证任何开奖结果。
